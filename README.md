# budgetapp
Mobile friendly budgeting app

This application will help users
- Determine their current financial snapshot
- Provide a debt replayment plan 
- Help allocate savings (Set percentages for budget)

Application Architecture 
- Front End (JavaScript, Reactive Native) 
- Back End (Python, Flask)
- Database (PostgreSQL)
- Hosting (Azure Functions)
- UI/UX (Figma, Material Design Framework)
